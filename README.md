# geo411-multiple-regression-models
Overview - Using R Studio to improve a multi variable regression model. 

Methods - Using r commands alter data analysis, and check diagnostic plots and summaries.

Key Findings - The base model had issues with minor multicollinearity and had skewed regression models for some variables such as living space in Sq.ft. I found logging and squaring variables can help solve some nonlinearity and improve the explanatory power of a model. Taking some variables out and adding some in also helped. 

Tools - R Studio
